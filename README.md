# Sistema Bancário em Java

Este projeto foi desenvolvido para praticar conceitos fundamentais da **Programação Orientada a Objetos (POO)** utilizando a linguagem **Java**.

O sistema simula um modelo simples de contas bancárias, permitindo trabalhar com diferentes tipos de conta como **Conta Corrente** e **Conta Poupança**.

---

## Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)

---

## Conceitos aplicados

Neste projeto foram utilizados diversos conceitos importantes de desenvolvimento em Java:

- Interface
- Classe abstrata
- Herança
- Polimorfismo
- Encapsulamento
- Enumeração (Enum)
- Sobrescrita de métodos (Override)

---

## Estrutura do projeto

Conta.java  
Interface que define os comportamentos que todas as contas devem possuir.

ContaBancaria.java  
Classe abstrata que contém os atributos e estrutura básica de uma conta bancária.

ContaCorrente.java  
Classe que representa uma conta corrente e implementa os métodos definidos pela interface.

ContaPoupanca.java  
Classe que representa uma conta poupança com regras específicas para depósitos.

TiposConta.java  
Enum que define os tipos de conta existentes no sistema.

---

## Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de:

- Praticar programação orientada a objetos
- Aprender a estruturar projetos em Java
- Simular um sistema bancário simples
